Welcome to the Toyota Explore Module Library

The Toyota Explore mobile app has a diverse range of interactive AR experiences. Most of these modules can be repurposed and adapted to create an experience tailored to the individual car. The module library has been created as an easy to use repository to both view the modules avaliable and also to understand how they function and look to the consumer.

Use the stacked menu in the top left to navigate to different sections of this library. Visit the modules page to see an overview of the wealth of experiences avaliable or visit an individual module type using the side menu. Each module page has a breakdown of the module including the module title, a video representation of the module & a list of cars which use this module. Some modules contain variations such as configurators and this is detailed with a breakdown of variations avaliable.
